What is Go?

    • Programming language created by Google and released in 2009
    • Motivated by their needs-
      • Efficency
      • Concurrency
      • Scalability
      (among other things)
    • Benefits of C/C++ without the frustrations

Why Go?

      Many reasons but here are a few:
    • Built in tools
      • go fmt for formatting
      • go doc for documentation
      • go test for testing
    • Garbage collection
    • Super fast and scalable
    • Not overly complex (no pointers, operator overloading, etc.)

What is Go good for?

    • Working with a lot of code and a lot of people
    • Large scale web apps
    • High performance systems
    • ...and more!

Goroutines

    • Great when you need concurrency- use Goroutines and channels
      • 
package main
import "fmt"
func f(from string) {
    for i := 0; i < 3; i++ {
        fmt.Println(from, ":", i)
    }
}
func main() {

	/*1*/f("blocks") //synchronous (blocking)
	/*2*/go f("goroutine") //goroutine- executes concurrently
	/*3*/go func(msg string) { //executes concurrently
		fmt.Println(msg)
	}('concurrent')
}

Channels

    • Channels connect concurrent goroutines.
    • Send values from one goroutine to another.
      • 
package main
import "fmt"

func main() {

	// Create a new channel
	messages := make(chan string)
	// Send a value into a channel
	go func() { messages <- "ping" }()
	// The <-channel receives a value
	msg := <-messages
	fmt.Println(msg)
}
